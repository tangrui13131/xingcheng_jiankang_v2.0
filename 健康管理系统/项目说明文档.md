# 我的星辰系统 (Enterprise Digital Ecosystem)

本项目是一个基于 [星辰 (XingChen-Vue)](https://gitee.com/y_project/XingChen-Vue) 框架深度定制和扩展的企业级数字化管理系统。它结合了现代化的前后端分离架构，并在原有系统的基础上集成了强大的**多模态 AI 助手**功能，致力于为企业提供高效、智能的业务管理生态。

---

## 🚀 核心特性

- **前后端分离架构**：前端采用现代化的 Vue 3 体系，后端采用经典的 Spring Boot 体系，保证了系统的高性能与可维护性。
- **丰富的内置功能**：继承了星辰框架的用户管理、部门管理、角色管理、菜单管理、字典管理、操作日志等基础权限与系统监控功能。
- **多模态 AI 助手 (新增)**：集成了基于 Kimi (Moonshot) 大模型 API 的智能助手。
  - **文本对话**：支持实时、连续的智能对话交互。
  - **图片分析**：支持上传图片，AI 可对图片内容进行视觉识别与分析。
  - **沉浸式 UI**：提供了类似主流 AI 产品的聊天气泡界面，支持打字机效果、自动滚动、快捷键发送等现代交互体验。

## 🛠️ 技术栈说明

### 后端 (XingChen-Vue)
后端是一个多模块的 Maven 项目，基于 Java 17 构建。
- **核心框架**: Spring Boot `4.0.3`
- **安全框架**: Spring Security & JWT Token
- **持久层框架**: MyBatis & PageHelper 分页插件
- **数据库连接池**: Alibaba Druid
- **缓存与中间件**: Redis
- **其他**: Fastjson2、Kaptcha (验证码)、Quartz (定时任务)、Oshi (系统监控)

#### 后端模块结构：
- `xingchen-admin`: 包含主启动类 `XingChenApplication` 和各个业务 Controller（包括新增的 `AiController`）。
- `xingchen-framework`: 包含系统核心配置、安全过滤器、异常处理、拦截器等。
- `xingchen-system`: 系统基础业务的 Service 和 Mapper 层（用户、角色、部门等）。
- `xingchen-common`: 包含各种通用工具类、常量、异常定义等。
- `xingchen-quartz`: 定时任务模块。
- `xingchen-generator`: 代码生成器模块。

### 前端 (XingChen-Vue3)
前端是一个基于 Vite 构建的单页面应用 (SPA)。
- **核心框架**: Vue `3.5.x` (Composition API & `<script setup>`)
- **构建工具**: Vite `6.4.x`
- **UI 组件库**: Element Plus `2.13.x`
- **状态管理**: Pinia `3.0.x`
- **路由管理**: Vue Router `4.6.x`
- **网络请求**: Axios

## 🤖 AI 助手功能详解

本系统的一大亮点是新增了智能 AI 助手模块，前后端协同完成了多模态的大模型接入：

1. **后端接入点**：
   - 路径：`xingchen-admin/src/main/java/com/xingchen/web/controller/al/AiController.java`
   - 接口：`POST /ai/chat/send`
   - 实现原理：接收前端传来的文本和 Base64 图片数据，按照 Moonshot (Kimi) 要求的格式组装 JSON 请求体，通过标准的 HTTP 请求调用 `https://api.moonshot.cn/v1/chat/completions` 接口，并将大模型的回复解析后返回给前端。

2. **前端交互层**：
   - 路径：`src/views/ai/chat/index.vue`
   - 实现原理：使用了 Vue 3 的响应式特性管理聊天记录数组。通过 Element Plus 组件实现了美观的对话框，包含气泡式消息、图片上传预览组件、状态加载动画以及输入框的快捷键绑定。

## 💻 本地运行指南

### 1. 环境准备
- **JDK**: Java 17
- **Node.js**: v18+ 推荐
- **数据库**: MySQL 8.x
- **缓存**: Redis

### 2. 启动后端服务
1. 将项目导入 IDEA 等 Java 开发工具中。
2. 确保本地 MySQL 和 Redis 服务已启动。
3. 如果是首次运行，请先导入相关的 SQL 初始化脚本（若有）。
4. 在 IDEA 中找到并运行 `E:\项目1\XingChen-Vue\xingchen-admin\src\main\java\com\xingchen\XingChenApplication.java`。
5. 看到控制台输出 `(♥◠‿◠)ﾉﾞ 星辰启动成功 ლ(´ڡ`ლ)ﾞ` 即表示启动成功（默认端口 `8080`）。

### 3. 启动前端服务
1. 打开终端，进入前端目录：`cd E:\项目1\XingChen-Vue3`
2. 安装依赖：`npm install`
3. 启动本地开发服务器：`npm run dev`
4. 启动后，终端会显示本地访问地址（如 `http://localhost:80/`），在浏览器中打开即可。
