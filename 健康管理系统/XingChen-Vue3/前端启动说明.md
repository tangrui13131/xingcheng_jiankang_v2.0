# 前端启动脚本说明

为了方便开发者快速启动前端项目，我们提供了专门的启动脚本。

## Windows 系统

双击运行 `启动前端.bat` 脚本，该脚本会自动完成以下操作：

1. 检查 Node.js 环境
2. 检查并安装包管理器 (yarn/npm)
3. 安装项目依赖
4. 检查后端服务状态
5. 启动前端开发服务器

## Linux/Mac 系统

在终端中执行以下命令：

```bash
chmod +x 启动前端.sh
./启动前端.sh
```

## 手动启动方式

如果您希望手动启动项目，可以使用以下命令：

```bash
# 安装依赖
yarn install

# 启动开发服务器
yarn dev
```

或使用 npm：

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 注意事项

1. 前端服务默认运行在 `http://localhost:80`
2. 前端需要后端服务支持，默认连接 `http://localhost:8080`
3. 首次启动需要下载依赖，可能需要较长时间
4. 确保端口未被其他程序占用
